# 01-goal.md — Цель

## Что делаем

Создать **работающий MVP** визуального AI-агента Руслан для Windows и macOS.

## Ключевые требования

1. **Кроссплатформенность** — Windows + macOS
2. **Модульная архитектура** — плагины, event bus, action engine
3. **LLM интеграция** — Гермес (Hermes/OpenAI API) выдаёт structured JSON
4. **Визуальный персонаж** — Godot 4 transparent overlay
5. **Голос** — Whisper (STT) + XTTS (TTS) + wake word
6. **Безопасность** — dry-run, подтверждения, лог

## Границы MVP (что НЕ делаем)
- 3D персонаж (только 2D)
- Multi-agent (Афина, Гефест)
- Онлайн-синхронизация
- Мобильное приложение

## Критерий готовности
- `pip install -e ".[dev]"` работает
- `python -m api.main` стартует без ошибок
- `pytest tests/ -v` — зелёные тесты
- macOS Controller создан и импортируется
- Godot проект открывается и подключается по WS
- PLAN.md актуален и размечен по фазам

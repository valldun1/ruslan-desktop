# 01-goal.md — цель фазового перехода

## Задача
Доделать 3 недостающие функции ruslan-desktop:

### Goal A: Голосовая активация (Voice Activation)
- Wake word detection (ключевое слово «Руслан»)
- Запись голоса с микрофона
- Speech-to-Text (faster-whisper или альтернатива)
- Отправка распознанного текста в Brain
- Индикация статуса (слушаю/думаю/говорю)

Критерий: сказать «Руслан, открой калькулятор» → открывается калькулятор

### Goal B: PNG-спрайт персонажа
- Создать PNG-спрайт по референсу (киберпанк-рыцарь, зелёное свечение)
- 6 состояний анимации: idle, thinking, speaking, happy, sad, working
- Интегрировать в ui/sprite_widget.py и ui/overlay.py
- Убрать эмодзи-заглушку

Критерий: оверлей показывает персонажа вместо 🛡

### Goal C: .dmg установщик
- Собрать macOS .dmg с приложением
- Включить Python рантайм + зависимости
- Автозапуск через launchd
- Иконка приложения

Критерий: скачать .dmg → перетащить в Applications → запустить Руслана
